# AI Movie Review Sentiment Analysis
A web-based machine learning application that predicts the sentiment of movie reviews as Positive or Negative using a deep learning model. The system uses an LSTM-based neural network trained on a movie review dataset and provides predictions through a simple web interface.
# Features
- Predicts sentiment of movie reviews

- Uses a deep learning model for text classification

- Web interface for user interaction

- Tokenization and sequence padding for text preprocessing

- Built using Python and Flask

- Easy to deploy and run locally
# Tech Stack
- Python
- Flask
- TensorFlow / Keras
- NumPy
- HTML
- CSS
